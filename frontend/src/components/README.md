# Components
Reusable React components organized by feature domain.
- auth/       Login, register, wallet connect components
- loan/       Loan cards, request form, repayment form, state badges
- dashboard/  Credit score gauge, loan overview cards
- credit/     Credit passport, history table, reputation score display
- audit/      Regulator portal components, Merkle verifier
- shared/     Buttons, modals, tables, inputs used across all features
